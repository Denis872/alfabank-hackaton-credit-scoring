## Инструкция по запуску проекта

Привет! В данном монорепозитории находятся проекты ML / Backend / Frontend. Ниже приведена иструкция работы с проектом:

1) Склонируйте репозиторий локально
2) Проверьте, установлен ли у вас Docker. Если нет, установите по ссылке https://www.docker.com/
3) В корне проекта запустите `sudo docker compose up --build -d`
4) Доджитесь сборки и загрузки контейнеров

Готово! Теперь доступ до приложений можно получить по следующим адресам:
1) Веб-приложение доступно по адресу `http://94.103.13.98:3000/`
2) Бекенд-приложение доступно по адресу `http://94.103.13.98:8080/predict/{id}`, где вместо {id} необходимо указать нужнный id пользователя
